# GrowSafe

## Project Description

GrowSafe is an intelligent garden protection system designed to detect and deter small animals from approaching plants. The goal of the project is to provide a simple, low-cost and effective solution for protecting garden areas without harming animals.

The system uses an ultrasonic sensor mounted on a servo motor to scan the surrounding area. When an object is detected within a defined distance (e.g. 50 cm), the system reacts using visual and acoustic feedback such as LEDs and a buzzer.

---

## How It Works

- The servo motor rotates the ultrasonic sensor (HC-SR04)
- The sensor measures distance continuously
- The system evaluates the measured distance
- If an object is detected within the threshold (e.g. 50 cm):
  - LED indicators are activated
  - Buzzer provides sound alerts
- If no object is detected, the system remains in safe mode

---

## Project Goal

The goal of GrowSafe is to protect plants from small animals using an automated, safe and environmentally friendly system.

---

## Components Used

- Arduino Uno  
- HC-SR04 Ultrasonic Sensor  
- Servo Motor  
- LED (Green / Blue / Red )  
- Buzzer  
- Breadboard and jumper wires
